# **airline-api**

An API that implements CRUD functionality for a fictional Airline by allowing the user to edit airplanes, Captains, and First Officers assigned to a scheduled flight.

**API Spec**

A PDF document containing the requirements for all endpoints and sample responses can be found in `api-spec.pdf` in the `spragubr_project` folder. 

**API URL**

https://spragubr-project-airline.appspot.com
